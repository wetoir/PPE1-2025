Miniprojet sur HTML

Création d’un site web : collecte, traitement et affichage des données

La création d'un site web

Ce miniprojet vise à faire un site web. On a fait d'abord un script qui collecte des données des sites données. Puis on crée un fichier .tsv pour stocker les nouvelles informations.

Après on a fait un site web pour montrer ces résultats sous forme d'un tableau. On modifie notre script pour faire cela automatiquement. Finalement, on déploie notre page GitHub en rajoutant un fichier index.html à la racine de notre GitHub.

Les étapes

Il consiste en 3 étapes :

  • Etape 1 : collecte des données sur les sites données
  • Etape 2 : représentation de données collectées sous forme d'un tableau d'un site web
  • Etape 3 : faire de sa page GitHub un site web qui dirige vers le site du tableau
